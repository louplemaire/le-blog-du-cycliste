- Install PHP and Mamp on your desktop

- Open Mamp -> go in ```Preferences``` -> ```Web Server```-> ```Select``` the project

- In Mamp page, click on ```My Website``` to open the blog
- Don't forget to click on ```Stop Server``` when leaving
